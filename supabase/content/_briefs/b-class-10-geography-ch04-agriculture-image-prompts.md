# Image prompt manifest — Class 10 Geography · Ch.4 Agriculture

All visuals in this bundle are **cowork-generated photos** (no inline SVG).
Each `media` turn in a lesson points at:
`…/storage/v1/object/public/lesson-images/b-class-10-geography-ch04-agriculture/<file>`

Workflow (same as Ch.3): the prompts below go to Drive
`cowork_images/class10-geo-ch04-course{1,2,3}.md`; cowork generates; operator
uploads the PNGs into the `lesson-images` bucket under the bundle-slug folder;
the predicted URLs then resolve. Images are **content-driven** — a lesson gets
an image only where a photo genuinely helps; count varies per lesson.

House style for every prompt: realistic documentary photograph, India context,
natural light, 16:9, **no text / no captions / no watermarks** in the image.

### `c1l1-slash-and-burn.png`

> late-afternoon light, hazy hills behind. Realistic, photojournalistic, 16:9,
> no text.

> hand-tended feel. Realistic, photojournalistic, 16:9, no text.
