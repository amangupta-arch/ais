# Image prompt manifest — Class 10 Geography · Ch.1

Generate each image (ChatGPT / DALL·E), **save it under the exact
`file:` name**, drop all of them in one folder, and push/hand them back.
I'll host them and insert the `media` turns at the noted placement — no
other edits needed.

## House style (keep the set cohesive)

- **Conceptual images → editorial illustration:** *"clean flat editorial
  vector illustration, muted earthy palette (ochre, moss green, slate
  blue, warm off-white background), soft shadows, simple shapes, friendly
  and modern, NO text, NO words, NO labels."*
- **Realism images → documentary photo:** *"realistic documentary
  photograph, natural daylight, Indian context, sharp focus, NO text, NO
  watermark."*
- Always append the **aspect ratio** and **"no text in the image"** —
  text inside generated images renders garbled and we add captions in the
  YAML anyway.
- For real historical events/people, prefer a *symbolic illustration*
  over a fake photoreal image (don't fabricate documentary photos of real
  events).

## How `file:` maps to the lesson
`<course>l<lesson>-<short>.png` → e.g. `c1l1-three-resources.png` belongs
to Course 1, Lesson 1. `placement:` says which turn it becomes.
