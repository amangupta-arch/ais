## Key client libraries

- `framer-motion` — `AnimatePresence` and layout animations in the lesson player.
- `lucide-react` — icon set used across the app.
- `next-mdx-remote` + `gray-matter` — wiki MDX rendering.
- `@anthropic-ai/sdk` — Claude API client (used server-side only).
- `@supabase/ssr` — Supabase client factory that respects request cookies.

## Server-side conventions

- Every route handler / page that talks to Supabase uses `createClient()` from `lib/supabase/server.ts` so auth context is propagated via the request cookie jar.
- Long-running routes set `export const maxDuration = 300;` (yaml-generate) or `30` (math-quiz) and `runtime = "nodejs"`.
- Sentry instrumentation lives in `instrumentation.ts` + `sentry.server.config.ts` + `instrumentation-client.ts`; everything no-ops cleanly when `NEXT_PUBLIC_SENTRY_DSN` is unset.
